CSS Gradient Examples

Visual examples of linear, radial, and conic gradients with full details.

Linear Gradient

background: linear-gradient(to right, #ff416c, #ff4b2b);

A linear gradient transitions colors along a straight line. You can specify the direction: to right, to bottom, 45deg, etc. You can add multiple color stops.

Linear Gradient with Angle

background: linear-gradient(135deg, #00c6ff, #0072ff);

Here, the gradient moves diagonally at a 135-degree angle from top left to bottom right.

Radial Gradient

background: radial-gradient(circle, #f7971e, #ffd200);

A radial gradient radiates from a central point. The shape can be a circle or ellipse. You can also specify position and multiple color stops.

Radial Gradient with Position

background: radial-gradient(circle at top left, #00f2fe, #4facfe);

This starts the radial gradient from the top left corner using circle at top left.

Conic Gradient

background: conic-gradient(from 0deg, red, yellow, green, blue, red);

A conic gradient rotates colors around a center point, like a pie chart. You can control the starting angle with from Xdeg. Not supported in older browsers like Internet Explorer.
