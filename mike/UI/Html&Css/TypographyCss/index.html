Typography in CSS

Font Families

The default font style of any text in browsers is Times New Roman. You can change it using the font-family property.

There are two main font families:

  • Serif: Fonts with small lines or decorations at the end of letters, like Times New Roman.
  • Sans-Serif: Fonts without these decorations, like Arial.

Example of a custom font family:

font-family: 'Gill Sana', sans-serif;

Text Shadow

The text-shadow property applies shadow to text. It takes four parameters:

  • h-offset: Horizontal offset (positive for right, negative for left).
  • v-offset: Vertical offset (positive for bottom, negative for top).
  • blur: How much blur you want.
  • color: The color of the shadow.

Example of a simple text shadow:

This is a basic text shadow example.
text-shadow: 5px 3px 5px blue;

Advanced Text Shadow Example

Advanced use of text-shadow allows multiple shadows for more complex effects. Here's an example:

This is an advanced text shadow example.
text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2), -2px -2px 4px rgba(0, 0, 0, 0.4);

This effect uses two shadows with different offsets and opacity levels to create a 3D-like appearance on the text.

Cursor in CSS

By default, browsers use two types of cursors:

  • Normal Cursor: The arrow cursor.
  • Pointer Cursor: The hand symbol cursor (used for links).

If you want to use a custom cursor, you can define it:

cursor: url('logo.png'), auto;

You can also change the cursor for the entire website using the universal selector:

* { cursor: url('cursor.png'), auto; }

Text Transformations

Text transformations are done using the text-transform property:

  • Uppercase Text: Makes all letters uppercase.
  • Underlined Text: Adds an underline to the text.
  • Line-through Text: Adds a line through the text.

Blockquote

Blockquotes are used for quoting text. They are often styled with indentation and italicized text:

Additional CSS Text Properties

Vertical Align

The vertical-align property aligns an element vertically:

vertical-align: top;

Font-Family

The font-family property is used to change the font type. Here's an example:

font-family: 'Gill Sana', sans-serif;

Text Shadow (Advanced Example)

The text-shadow property allows you to apply multiple shadows to create more complex effects:

text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.2), -2px -2px 4px rgba(0, 0, 0, 0.4);

This example creates a layered shadow effect:

  • The first shadow is offset by 2px to the right and 2px down, with a blur and lighter opacity.
  • The second shadow is offset by 2px to the left and 2px up, with a blur and darker opacity.

This creates a 3D effect on the text.
