As the name suggests, these sizes stay fixed on all screens. For example, setting 22px font size will look the same on desktops, tablets, and mobiles.

  • Inch (in)
  • Centimeter (cm)
  • Millimeter (mm)
  • Pixel (px)
  • Point (pt)
  • Pica (pc)

Relation between fixed sizes:

1in = 2.54cm = 25.4mm = 96px = 72pt = 6pc

Note: Inches and cm can be imprecise on screens; pixels (px) are preferred in web development.
