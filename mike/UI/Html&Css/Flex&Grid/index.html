9Ô∏è‚É£ What is CSS Grid?

CSS Grid is a two-dimensional layout system that arranges content in rows and columns.

Parent Property: display: grid

üîü Defining Columns & Rows

Use grid-template-columns and grid-template-rows to define grid tracks:


.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

1Ô∏è‚É£1Ô∏è‚É£ Grid Lines & Spanning

Every grid cell has horizontal and vertical lines. You can position and size items by specifying which grid lines they start and end at using grid-column and grid-row. 


.item {
  grid-column: 1 / 3;  /* Spans from column line 1 to line 3 */
  grid-row: 1 / 2;     /* Spans from row line 1 to line 2 */
}

Alternatively, use the shorthand grid-column: span 2;

1Ô∏è‚É£2Ô∏è‚É£ Grid Auto Rows/Columns

When the grid creates new tracks automatically (e.g., when items overflow), you can control their size using: 


.container {
  grid-auto-rows: 100px;
  grid-auto-columns: 1fr;
}

This ensures all implicitly created rows or columns follow a consistent size rule.

1Ô∏è‚É£3Ô∏è‚É£ Grid Area & Named Areas

You can define areas using grid-area and name them for easy layout using grid-template-areas. 


.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar content";
  grid-template-columns: 1fr 3fr;
}
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }

This makes complex layouts easy to read and manage.
